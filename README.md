# 🔐 Auth App

A simple and responsive authentication app built with vanilla JavaScript.

---

## 🚀 Features
- User signup
- User login
- Logout functionality
- Multiple users support
- Inline validation (no alerts)
- Persistent login using LocalStorage
- Dark mode toggle 🌙
- Enter key support for form submission

---

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- LocalStorage
